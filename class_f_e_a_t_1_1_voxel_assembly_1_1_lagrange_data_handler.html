Detailed Description

template<typename Space_, typename DT_, typename IT_>
class FEAT::VoxelAssembly::LagrangeDataHandler< Space_, DT_, IT_ >

Data handler for Lagrange based FE spaces.

This class handles the data extraction of the underlying mesh, space and color data for voxel assembly.

Note
Only works for Lagrange space because it is implicitly required that the mapping of the first k dofs are matched with the actual node numeration.
Template Parameters
Space_The underlying space type.
DT_The datatype.
IT_The indextype.
