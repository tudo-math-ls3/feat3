Detailed Description

template<typename DT_, typename IT_, template< typename, typename > class Functional_>
class FEAT::Control::Meshopt::NonlinearSystemLevel< DT_, IT_, Functional_ >

(Non)linear system of equations on one mesh refinement level

Template Parameters
DT_Floating point type
IT_Index type
Functional_The (patch-) local nonlinear, nonquadratic mesh quality functional

Since the mesh quality functional is not quadratic, the gradient is a nonlinear mapping.
