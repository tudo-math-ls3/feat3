Detailed Description

template<typename DomainLevel_, typename TemplateSet_>
class FEAT::Control::Domain::AdaptiveLevelWrapper< DomainLevel_, TemplateSet_ >

Wrapper class for domain levels for AdaptivePartiDomainControl.

Wrapper class that extends the given DomainLevel class by an additional mesh_layer member, that gives access to the refinement tree of the adaptive mesh created by the AdaptivePartiDomainControl.
