Detailed Description

template<typename DT_, typename IT_, int BlockSize_>
class FEAT::LAFEM::DenseVectorBlocked< DT_, IT_, BlockSize_ >

Blocked Dense data vector class template.

Template Parameters
DT_The datatype to be used.
IT_The indextype to be used.
BlockSize_The size of the represented blocks

This class represents a vector of continuous data in memory.
Logical, the data are organized in small blocks of BlockSize_ length.

Data survey:
_elements[0]: raw number values
_scalar_index[0]: container size - aka block count _scalar_index[1]: boolean flag, signaling DenseVectorRange usage

Refer to LAFEM design description for general usage informations.

Author
Dirk Ribbrock
