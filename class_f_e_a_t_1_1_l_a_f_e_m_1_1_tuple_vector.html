Detailed Description

template<typename First_, typename... Rest_>
class FEAT::LAFEM::TupleVector< First_, Rest_ >

Variadic TupleVector class template.

This class template implements a composition of sub-vectors of arbitrary classes.

Template Parameters
First_,...Rest_A sequence of (meta) vector classes which are to be composed.
Author
Peter Zajac
