Detailed Description

template<typename SubMirror_, int count_>
class FEAT::LAFEM::PowerMirror< SubMirror_, count_ >

PowerVector meta-mirror class template.

This class template implements a vector-mirror which is capable of mirroring PowerVector objects.

Attention
This class template uses the same mirror for all blocks of a PowerVector object!
Template Parameters
SubMirror_The type of the sub-mirror.
count_The number of sub-mirror blocks.
Author
Peter Zajac

Updates a scatter mask vector for this mirror.

This function is used to create a so-called "scatter mask vector", i.e. a vector, in which each entry, that belongs to a DOF that is shared with other processes, is set to 1, and all other entries are set to 0 (or vice versa). This mask vector can then be used to determine the number of local and/or shared DOFs for this process.

Parameters
[in]vectorA template vector to be used to determine the POD size; must be allocated to correct size but its contents are ignored.
[in,out]maskThe mask vector to be updated.
[in]valueThe value that is to be used to set the mask vector entries referenced by this mirror.
[in]offsetThe offset of the first native/POD DOF in the mask vector.
Returns
The size of the input template vector.
