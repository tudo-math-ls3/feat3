Computes a quality indicator concerning the cell sizes.

Parameters
[out]lambda_minMinimum of the optimal cell size lambda over all cells
[out]lambda_maxMaximum of the optimal cell size lambda over all cells
[out]vol_minMinimum cell volume
[out]vol_maxMaximum cell volume
[out]volTotal volume of the domain given by the mesh

In a truly optimal mesh (consisting ONLY of reference cells of the right size), every cell's volume is exactly lambda(cell). This is especially the goal for r-adaptivity. So in an optimal mesh,

\[ \forall K \in \mathcal{T}_h: |K|/|\Omega| = \lambda(K) \]

so we compute the 1-norm of the vector \((v)_i = \left| \frac{|K_i|}{\sum_j |K_j|} - \lambda(K_i) \right| \).

Returns
The relative cell size quality indicator.
Note
lambda_min, lambda_max, vol_min, and vol_max are all volume fractions.
As these quantities are global, this function has a pre_sync and post_sync part.
See also
