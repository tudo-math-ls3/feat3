Detailed Description

template<typename Vector_, typename Space_, int max_der_>
class FEAT::Assembly::TraceAssemblyDiscreteFunctionIntegralJob< Vector_, Space_, max_der_ >

Assembly job for the trace integration of a discrete finite element function.

Template Parameters
Vector_The vector that represents the discrete function to be integrated.
Space_The finite element space that is to be used for the discretization.
max_der_The maximum derivative of the function that is to be integrated.
