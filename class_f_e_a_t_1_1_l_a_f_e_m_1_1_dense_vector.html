Detailed Description

template<typename DT_, typename IT_ = Index>
class FEAT::LAFEM::DenseVector< DT_, IT_ >

Dense data vector class template.

Template Parameters
DT_The datatype to be used.
IT_The indextype to be used.

This class represents a vector of continuous data in memory.

Data survey:
_elements[0]: raw number values
_scalar_index[0]: container size

Refer to LAFEM design description for general usage informations.

Author
Dirk Ribbrock
