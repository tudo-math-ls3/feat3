Detailed Description

template<typename DataType_, typename Function_, typename Trafo_, int max_der_>
class FEAT::Assembly::AnalyticFunctionIntegralJob< DataType_, Function_, Trafo_, max_der_ >

Assembly job for the integration of an analytic function.

Template Parameters
DataType_The (scalar) datatype in which the assembly is to be performed.
Function_The analytic function class that is to be integrated.
Trafo_The trafo on which to integrate.
max_der_The maximum derivative order of the integrals that are to be assembled.
