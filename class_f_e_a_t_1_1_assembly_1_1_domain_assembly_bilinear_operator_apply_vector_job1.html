Detailed Description

template<typename BilinearOperator_, typename Vector_, typename VectorSol_, typename Space_>
class FEAT::Assembly::DomainAssemblyBilinearOperatorApplyVectorJob1< BilinearOperator_, Vector_, VectorSol_, Space_ >

Vector assembly job for BilinearOperator and identical test-/trial-spaces.

This class implements the DomainAssemblyJob interface to assemble a vector that is calculated by

\( \int_\Omega \tilde{v} \psi = \int_\Omega A(v, \psi) dx, m=1,\dots,d \) where \( A(\cdot, \cdot) \) is provided by a BilinearOperator interface and \(v\) is a finite element vector.

Template Parameters
BilinearOperator_The bilinear operator that is to be assembled
Vector_The vector that is to be assembled.
VectorSol_The vector that the biliniear operator is to be applied to.
Space_The finite element space to be used as test- and trial-space.
