Detailed Description

template<typename DataType_, int dim_>
class FEAT::Analytic::Common::PoiseuillePipeFlow< DataType_, dim_ >

Parabolic Poiseuille Pipe-Flow velocity field.

This function is effectively a 3D equivalent of the ParProfileVector function, which represents a parabolic flow inside a 3D cylinder. This function is an analytic steady state solution to the 3D incompressible Navier-Stokes equations in a cylinder domain (with constant radius) and thus represent a Poiseuille flow in this domain.

This function is parameterized in the rotational axis and the radius of the cylinder domain as well as the maximum flow velocity in the center of the cylinder.
