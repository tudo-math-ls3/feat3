Detailed Description

template<typename Shape_, int num_coords_, typename Coord_>
class FEAT::Geometry::Parti2Lvl< ConformalMesh< Shape_, num_coords_, Coord_ > >

2-Level-Partitioner class template specialization for ConformalMesh

This class implements a 2-Level-Partitioner, i.e. a simple partitioner that exploits the 2-level refinement algorithm and numbering.

The 2-Level-Partitioner can provide a valid partitioning of a (refined) mesh, if the number of required patches matches the number of elements in a refined mesh.

For Hypercube meshes, this algorithm can do even better: Let n be the number of elements in the mesh and p the desired number of patches, then this algorithm can provide a valid partitioning if \(p = n\cdot2^k\) for some \(k\in\mathbb{N}\).

The basic usage of this class is as follows:

  1. Create an object of this class and pass the to-be-partitioned mesh as well as the desired number of ranks/patches to the constructor.
  2. If the success() function returns true, then a valid 2-level partitioning has been found. If the function returns false, then no 2-level partitioning exists and one has to try some other partitioning strategy.
  3. Refine the mesh up to the required refinement level returned by the parti_level() function.
  4. Create the Elements-At-Rank graph using the build_elems_at_rank() function.
