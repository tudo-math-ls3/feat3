Detailed Description

template<typename Mesh_>
class FEAT::Geometry::Atlas::Circle< Mesh_ >

Circle chart class template.

This class represents a 2D circle characterized by a midpoint, a radius and a 1D interval for the parameter space.

This chart implements both the implicit and explicit chart interfaces.

The parameter domain [a,b] is mapped onto the circle, where both interval endpoints are mapped onto the east point on the circle, i.e. the point with the X/Y-coordinates [mid_x + radius, mid_y]. If the parameter interval is oriented positive, i.e. if it holds that a < b, then the circle is parameterized in counter-clock-wise orientation. If the parameter interval is oriented negative, i.e. if it holds that b < a, then the circle is parameterized in clock-wise orientation.

Template Parameters
Mesh_The type of the mesh to be parameterized by this chart.

Applies a "proper rigid" transformation onto the chart.

Let v denote the origin world point, w the offset world point and R the rotation matrix corresponding to the angles, then this function applies the following transformation for any point x of the chart:

Parameters
[in]originThe origin of the transformation. This is subtracted from any point before applying the rotation.
[in]anglesThe angles of the rotation matrix.
  • 2D: the rotation angle in radians is stored as:
    • angles(0): rotation angle
    • angles(1): -ignored-
  • 3D: the rotation angles in radians stored as:
    • angles(0): yaw angle
    • angles(1): pitch angle
    • angles(2): roll angle
[in]offsetThe offset of the transformation. This is added to any point after applying the rotation.
