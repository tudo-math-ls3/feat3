Detailed Description

template<typename Function_, typename Vector_, typename Space_, int max_der_>
class FEAT::Assembly::TraceAssemblyErrorFunctionIntegralJob< Function_, Vector_, Space_, max_der_ >

Assembly job for the trace integration of a analytic vs discrete error function.

This class is used to integrate the error function (u - u_h), where u is an analytic function and u_h is a discrete finite element function.

Template Parameters
Function_The type of the analytic function u that is to be integrated.
Vector_The type of the coefficient vector of the discrete function u_h. May be one of the following:
  • LAFEM::DenseVector<...>
  • LAFEM::DenseVectorBlocked<...>
Space_The finite element space to be used for the discretization
max_der_The maximum derivative of the function that is to be integrated.
