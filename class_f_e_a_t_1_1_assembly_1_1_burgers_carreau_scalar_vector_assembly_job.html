Detailed Description

template<typename Vector_, typename Space_, typename ConvVector_>
class FEAT::Assembly::BurgersCarreauScalarVectorAssemblyJob< Vector_, Space_, ConvVector_ >

Burgers Carreau assembly job for scalar right-hand-side vector.

This assembly job implements the scalar Burgers operator assembly, which is assembled into a right-hand-side vector.

See the documentation of the BurgersAssemblyJobBase class template (which is a base class of this class) for details about the terms and coefficients offered by this assembly job.

Template Parameters
Vector_The type of the scalar vector that is to be assembled.
Space_The finite element space to be used for assembly
ConvVector_The type of the convection (velocity) vector field.

Maximilian Esser
