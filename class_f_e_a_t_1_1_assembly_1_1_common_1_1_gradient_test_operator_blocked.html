template<int dimension_>
class FEAT::Assembly::Common::GradientTestOperatorBlocked< dimension_ >

Gradient operator applied onto the test function.

This is the gradient bilinear form using blocked types with partial integration: \( a(\phi, \psi)_m = \int_\Omega(\phi e_m, \nabla \psi) dx, m=1,\dots,d \)
