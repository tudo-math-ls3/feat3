Detailed Description

template<typename DomainLevel_>
class FEAT::Control::Domain::VoxelDomainLevelWrapper< DomainLevel_ >

Wrapper class for domain levels for VoxelDomainControl.

This class acts as a derived class for a domain level class and extends it by an additional "slag level" object, which is required to assemble grid transfer operators (i.e. prolongation/restriction) for spaces organized in the VoxelDomainControl class. Furthermore, this class also stores the optional element coloring and element layering for parallel assembly.
