Detailed Description

template<typename Mesh_, typename Chart_ = Atlas::ChartBase<Mesh_>>
class FEAT::Geometry::ChartHitTestFactory< Mesh_, Chart_ >

Chart-based Hit-Test Factory class template.

This class template can be used to create a MeshPart for a particular mesh, which consists of all entities that are inside or outside the region characterized by a given chart.

Template Parameters
Mesh_The type of the mesh for which the cell sub-set is to be computed.
Chart_The type of the chart that is to be tested; typically Atlas::ChartBase.
Author
Peter Zajac, Stefan Wahlers
