Detailed Description

template<typename DT_, typename IT_, typename DomainControl_, template< typename, typename, typename > class Hyperelasticity_>
class FEAT::Control::Meshopt::HyperelasticityFunctionalControl< DT_, IT_, DomainControl_, Hyperelasticity_ >

Control class for HyperelasticityFunctionals.

Template Parameters
DT_The floating point precision for the solver
IT_Index type
DomainControl_The domain control type this is based on
Trafo_The mesh's underlying transformation. At the time of writing, there is only Trafo::Standard, which means P1/Q1 transformation.
Hyperelasticity_The (patch-) local HyperelasticityFunctional to use.
See also
FEAT::Meshopt::HyperElasticityFunctionalBase
Author
Jordi Paul
