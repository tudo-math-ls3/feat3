Detailed Description

template<typename Coord_>
class FEAT::Geometry::MeshPartExtruderFactory< ConformalMesh< Shape::Hypercube< 2 >, 2, Coord_ > >

MeshPart Extruder factory.

This class implements the Factory interface using a MeshExtruder object to extrude a given 2D quadrilateral mesh-part to a 3D hexahedral mesh-part.
