Detailed Description

template<typename Filter_, typename DT_, typename IT_>
class FEAT::Solver::SSORPrecondWithBackend< PreferredBackend::generic, LAFEM::SparseMatrixCSR< DT_, IT_ >, Filter_ >

SSOR preconditioner internal implementation.

This class implements a simple SSOR preconditioner, e.g. zero fill-in and no pivoting.

This implementation works for the following matrix types and combinations thereof:

  • LAFEM::SparseMatrixCSR

Moreover, this implementation supports only Mem::Main
