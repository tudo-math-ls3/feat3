Detailed Description

template<typename MatrixA_, typename MatrixB_ = MatrixA_, typename MatrixD_ = MatrixB_>
class FEAT::LAFEM::SaddlePointMatrix< MatrixA_, MatrixB_, MatrixD_ >

Saddle-Point matrix meta class template.

This class template implements a meta container for saddle-point matrices: 

/ A B \
\ D 0 /

where

  • A is an n-by-n (meta) matrix
  • B is an n-by-m (meta) matrix
  • D is an m-by-n (meta) matrix
Template Parameters
MatrixA_The type of the upper-left (square) matrix block A.
MatrixB_The type of the upper-right (rectangular) matrix block B.
MatrixD_The type of the lower-left (rectangular) matrix block D.
Author
Peter Zajac
