Detailed Description

template<typename V, typename Key = SlotMapKey>
class FEAT::Util::SlotMap< V, Key >::SlotMapIterator

Iterator type for the SlotMap.

Iterates over the data array in order. Returns pairs consisting of references to the data and the corresponding keys.

Note that this provides a more idiomatic api than forwarding the std::vector iterator. Doing so would expose a pointer into the data array to the user and allow access via indices, bypassing the slotmaps keys. The user can still retrieve a pointer from the passed out references, but doing so is a more obvious code smell.
