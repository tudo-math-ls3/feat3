Detailed Description

Bilinear Operator Assembler class template.

This class template implements the assembly of a standard bilinear operator into a matrix.

inlinestatic

Applies the bilinear operator to an FE function.

This function is the version for identical test- and trial-spaces and does the same as assembling the operator into a matrix and multiplying it with a vector representing the FE function. Useful if this is needed only a few times and one does not want to save the matrix.

Parameters
[out]retThe transient return vector.
[in]coeff_vectorThe transient coefficient vector for the FE function.
[in]operatA transient reference to the operator implementing the BilinearOperator interface to be assembled.
[in]spaceA transient reference to the finite-element to be used as the test- and trial-space.
[in]cubature_factoryA transient reference to the cubature factory to be used for integration.
[in]alphaThe scaling factor for the bilinear operator.
