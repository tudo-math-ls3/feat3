Detailed Description

CuSolverLU solver class.

This class provides an implementation of the SolverBase interface using the direct cuda solver cusolverSpDcsrlsvluHost for doing the actual dirty work.

Attention
This class is only declared if FEAT was configured to build and link against cuda and the cuda sdk ships with the cusolver library (cuda version >= 7).
