Detailed Description

template<typename Shape_, int num_coords_, typename Coord_>
class FEAT::Geometry::PatchMeshPartSplitter< Geometry::ConformalMesh< Shape_, num_coords_, Coord_ > >

Class for creating MeshParts referring to a patch BaseMesh, ConformalMesh specialization.

Template Parameters
Shape_Shape type of the mesh cells
num_coords_World dimension
Coord_Floating point type for vertex coordinates

Like the PatchPartMap and PatchPartMapHolder, this assembles information about the BaseMesh and PatchMeshPart first and can later be used to create several MeshParts referring to the PatchMeshPart by calling build() and then the Factory interface, one at a time.
