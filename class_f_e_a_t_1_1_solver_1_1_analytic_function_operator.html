Detailed Description

template<typename DT_, typename IT_, typename Function_>
class FEAT::Solver::AnalyticFunctionOperator< DT_, IT_, Function_ >

Wrapper class defining an operator based on a scalar AnalyticFunction.

Template Parameters
DT_Floating point precision
IT_Index type
Thescalar AnalyticFunction

Since all solvers work on LAFEM containers, this is hardwired to expect LAFEM containers and just use the first entry, i.e. a scalar function with 3 input parameters results in a LAFEM::DenseVectorBlocked with one entry of BlockSize 3.

Because AnalyticFunctions work in Mem::Main, setting Mem_ to anything else will greatly slow things down due to excessive up-/downloading. It's still here to provide a unified class interface for operators.
