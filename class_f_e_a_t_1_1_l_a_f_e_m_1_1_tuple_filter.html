Detailed Description

template<typename First_, typename... Rest_>
class FEAT::LAFEM::TupleFilter< First_, Rest_ >

TupleVector meta-filter class template.

This class template implements a composition of sub-filters of arbitrary classes, which can be applied onto TupleVector meta-container objects.

Note
If you are looking for a way to define a filter which is a composition of filters successively applied onto the same vector, see the FilterChain class template.
Template Parameters
First_,Rest_...The filters to be composed.
