Detailed Description

template<typename ElementalFunction_, typename Trafo_, typename RefCellTrafo_ = RumpfTrafo<Trafo_, typename Trafo_::CoordType>>
class FEAT::Meshopt::MeshConcentrationFunction< ElementalFunction_, Trafo_, RefCellTrafo_ >

Class to compute a desired concentration for the mesh cell distribution.

Template Parameters
ElementalFunction_The scalar function that computes the concentration.
Trafo_Our transformation.
RefCellTrafo_Mesh optimization reference cell transformation.
Note
To avoid virtual functions, this class explicitly knows which type of elemental function it has.

The ElementalFunction_ is needed to compute grad_h etc., but we do not need to know from what quantity it maps to the concentration. This is only needed in classes derived from this.

Computes the local gradient of the optimal scales.

Parameters
[out]sum_detThe sum of all optimal reference cell's volumes, which is different from the domain's volume in general.

The optimal scales \( h \) depend on the concentration \( c \) by the relation

\[ \forall K_k \in \mathcal{T}_h: h_k = \left( \frac{c (K_k)}{\sum_{l=1}^N c(K_l) \sum_{l=1}^N \mathrm{det} \nabla R_{T,l}(\phi)} \right)^{\frac{1}{d}} \]

The concentration function in turn indirectly depends on the vertex locations \( \{ x_i : i=1, \dots, n \} \) via

\[ c(K_k) = (\alpha + \sum_{x_j \in K_k} \varphi(x_j))^\beta, \]

so that we have to take this dependency into account for the full gradient. Define

\[ s_d := \sum_{l=1}^N \mathrm{det} \nabla R_{T,l}(\Phi), s_c := \frac{c (K_k)}{\sum_{l=1}^N c(K_l)}. \]

So for each \( x_j \) we arrive at

\[ \frac{\partial h(K_k)}{\partial x_j} = \frac{1}{d} \left( \frac{c(K_k)}{s_d s_c} \right)^ {\frac{1}{d}-1} \left[ c(K_k) ( \frac{\partial s_d}{\partial x_j} s_c + s_d \frac{\partial s_c}{\partial x_j} ) + \frac{\partial c(K_k)}{\partial x_j} s_d s_c \right] (s_d s_c)^{-2} \]
