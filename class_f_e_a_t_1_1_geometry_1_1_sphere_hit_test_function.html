Detailed Description

template<typename DataType_, int dim_>
class FEAT::Geometry::SphereHitTestFunction< DataType_, dim_ >

Sphere hit-test function.

This function implements a spherical hit-test, i.e. it checks whether a point is inside or outside of a sphere.

Template Parameters
DataType_The data-type to be used by the function.
dim_The dimension of the sphere.
