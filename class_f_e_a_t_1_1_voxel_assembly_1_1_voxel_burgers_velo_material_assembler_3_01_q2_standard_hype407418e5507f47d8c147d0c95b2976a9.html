Detailed Description

template<int dim_, typename DT_, typename IT_>
class FEAT::VoxelAssembly::VoxelBurgersVeloMaterialAssembler< Q2StandardHyperCube< dim_ >, DT_, IT_ >

Q2Lagrange thread parallel assembly class for the burgers velo material operator.

This class provides host and device variants for a thread parallel assembly of the extended burgers operator. For details on the burgers velocity material operator, see the burgers velocity material assembler in </kernel/assembly/>. The threading strategy is based on a cell based coloring of the mesh, so in principle this version for Q2 space works as long as a valid coloring is provided.

Template Parameters
dim_Dimension of the hypercubes.
DT_Datatype to be used.
IT_Indextype to be used.
