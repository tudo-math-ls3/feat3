Detailed Description

Du:Dv operator implementation.

This class implements the weak formulation of the bilinear Du : Dv operator, i.e.

\[ \mathbf{D} \varphi : \mathbf{D} \psi \]

with the operator

\[ \mathbf{D} \varphi = \frac{1}{2} \left( \nabla \varphi + \left( \nabla \varphi \right)^T \right). \]

Note that

\[ \mathbf{D} \varphi : \mathbf{D} \psi = 2 \left( \nabla \varphi : \nabla \psi + \nabla \varphi : \left( \nabla \psi \right)^T \right), \]

so the \( (k,l) \)-block consists of entries corresponding to \( \partial_k \varphi \partial_l \psi \).

This class can be used with the BilinearOperator assembly class template to assemble one scalar matrix corresponding to one block of the \( d \times d \) block matrix, where \( d \) is the number of space dimensions.
