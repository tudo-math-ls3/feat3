Detailed Description

template<typename DT_, typename IT_>
struct FEAT::Control::Meshopt::ControlFactory< DT_, IT_ >

Factory for MeshoptControl objects.

Template Parameters
DT_Floating point type for the solver.
IT_Index type for the solver.

This class can construct MeshoptControl objects of different types at runtime based on PropertyMaps and returns them through base class std::shared_ptrs.

Note
At this time, the only transformation available is Trafo::Standard.
