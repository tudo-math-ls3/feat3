Detailed Description

template<typename Derived_, typename Matrix_, typename TestSpace_, typename TrialSpace_, TrafoTags trafo_config_, SpaceTags test_config_, SpaceTags trial_config_>
class FEAT::Assembly::BasicMatrixAssemblyTaskCRTP2< Derived_, Matrix_, TestSpace_, TrialSpace_, trafo_config_, test_config_, trial_config_ >

Basic Matrix assembly task CRTP base-class for (possibly different) test-/trial-spaces.

This CRTP class can be used as a base-class for various types of matrix assemblies tasks. This class implements all the five interface member functions required by the DomainAssemblyJob interface. The assemble member function calls the two functions set_point and eval via static down-cast, so these two functions have to be provided by the derived class.

Template Parameters
Derived_The most derived CRTP class
Matrix_The type of the matrix that is to be assembled
TestSpace_The finite element space to be used as test-space
TrialSpace_The finite element space to be used as trial-space
trafo_config_The trafo configuration tags required for the assembly
test_config_The test-space configuration tags required for the assembly
trial_config_The trial-space configuration tags required for the assembly
