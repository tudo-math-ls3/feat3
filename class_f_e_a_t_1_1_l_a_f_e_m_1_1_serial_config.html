Detailed Description

Config class for serialize parameter.

Data survey: elements_compression LAFEM::SerializeMode, The compression mode for the elements array. Refer to CompressionModes for details. indices_compression LAFEM::SerializeMode, The compression mode for the indices array tolerance FEAT::Real, The max error that should be occur while compressing with zfp.

Note
tolerance should be unset(meaning -1.) if elements_compression != CompressionModes::elements_zfp and strictly greater 0 if elements_compression == CompressionModes::elements_zfp.
the corresponding configure flags 'zlib' and/or 'zfp' need to be added in the build-id at the configure call.
