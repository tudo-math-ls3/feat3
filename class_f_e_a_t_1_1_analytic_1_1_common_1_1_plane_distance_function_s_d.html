Detailed Description

template<int component_, int dim_, typename DataType_>
class FEAT::Analytic::Common::PlaneDistanceFunctionSD< component_, dim_, DataType_ >

Scaled and displaced analytic distance from i-th coordinate axis/plane function.

This class implements the AnalyticFunction interface representing the scaled and displaced distance function \( f(x) = b | (x - x_0)_i | \)

This class supports function values and gradients for all dimensions.

Template Parameters
component_Index of the coordinate axis/plane.
ImgPointType_The type of \( x_0 \).
