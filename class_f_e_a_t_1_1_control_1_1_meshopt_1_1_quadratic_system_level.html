Detailed Description

template<typename DT_, typename IT_, template< typename, typename > class Functional_>
class FEAT::Control::Meshopt::QuadraticSystemLevel< DT_, IT_, Functional_ >

SystemLevel for a quadratic mesh quality functional leading to a linear system.

Template Parameters
DT_Floating point type
IT_Index type
Functional_The (patch-) local quadratic mesh quality functional

Since the mesh quality functional is quadratic, it can be minimized by solving the the linear system of equations given by its gradient, represented by the GlobalSystemMatrix.
