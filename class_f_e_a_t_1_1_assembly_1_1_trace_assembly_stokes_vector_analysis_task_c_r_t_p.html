Detailed Description

template<typename Derived_, typename VectorVelo_, typename VectorPres_, typename SpaceVelo_, typename SpacePres_, TrafoTags trafo_config_, TrafoTags facet_trafo_config_, SpaceTags space_velo_config_, SpaceTags space_pres_config_>
class FEAT::Assembly::TraceAssemblyStokesVectorAnalysisTaskCRTP< Derived_, VectorVelo_, VectorPres_, SpaceVelo_, SpacePres_, trafo_config_, facet_trafo_config_, space_velo_config_, space_pres_config_ >

Basic Stokes Vector analysis task CRTP base-class.

This class can be used as a base class for some sort of post-processing assembly of a Stokes solution vector. The most prominent use case of this class is the computation of body forces on an obstacle.

Template Parameters
Derived_The most derived CRTP class
VectorVelo_The type of the velocity vector that is to be assembled
VectorPres_The type of the pressure vector that is to be assembled
SpaceVelo_The finite element velocity space
SpacePres_The finite element pressure space
trafo_config_The cell trafo configuration tags required for the assembly
facet_trafo_config_The facet trafo configuration tags required for the assembly
space_velo_config_The velocity space configuration tags required for the assembly
space_pres_config_The pressure space configuration tags required for the assembly
Note
This class (ab)uses the base class for different trial/test functions to define the assembly traits and all the required helper structures for the velocity/pressure finite element space pair.
Author
Peter Zajac
