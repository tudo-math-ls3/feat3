Detailed Description

template<typename FirstRow_, typename... RestRows_>
class FEAT::LAFEM::TupleMatrix< FirstRow_, RestRows_ >

Variadic TupleMatrix class template.

Template Parameters
FirstRow_,...RestRows_A sequence of TupleMatrixRow template class instances which are to be composed
Attention
All template parameters supplied to this class must be instances of the TupleMatrixRow class template, even if the TupleMatrix only has 1 meta-column!
Author
Peter Zajac
