Detailed Description

template<typename LumpedMatrixA_, typename MatrixB_, typename MatrixD_, typename FilterA_>
class FEAT::Global::SymmetricLumpedSchurMatrix< LumpedMatrixA_, MatrixB_, MatrixD_, FilterA_ >

Symmetric lumped Schur complement matrix.

Template Parameters
LumpedMatrixA_Type for the vector representing the diagonal middle matrix
MatrixB_Type for the right matrix
MatrixD_Type for the left matrix
FilterA_Filter for the diagonal middle matrix

For given matrices \( B, A = diag(a), D = B^T \), this emulates the matrix

\[ S = B^T A^{-1} B. \]

This is called Lumped because the diagonal structure of A usually comes from lumping.

Warning
For efficiency reasons, \( B^T \) needs to be explicitly computed and passed as \( D \), but this property is only checked very superficially (and only in debug mode).
Note
In the current implementation, all container template parameters must be Global:: as the synchronization is done manually by using local() objects and calling sync_0()
Author
Jordi Paul
