Detailed Description

template<typename SubType_, int blocks_>
class FEAT::LAFEM::PowerColMatrix< SubType_, blocks_ >

Power-Col-Matrix meta class template.

This class template implements a vertical composition of n sub-matrices of the same class. This can be interpreted as a dense m-by-1 matrix of other matrices.

Template Parameters
SubType_The type of the sub-matrix.
blocks_The number of sub-matrix blocks.
Author
Peter Zajac
