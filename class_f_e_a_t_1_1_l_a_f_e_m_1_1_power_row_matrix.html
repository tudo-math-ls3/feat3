Detailed Description

template<typename SubType_, int blocks_>
class FEAT::LAFEM::PowerRowMatrix< SubType_, blocks_ >

Power-Row-Matrix meta class template.

This class template implements a horizontal composition of n sub-matrices of the same class. This can be interpreted as a dense 1-by-m matrix of other matrices.

Template Parameters
SubType_The type of the sub-matrix.
blocks_The number of sub-matrix blocks.
Author
Peter Zajac
