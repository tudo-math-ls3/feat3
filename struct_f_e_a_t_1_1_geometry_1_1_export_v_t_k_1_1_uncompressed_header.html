Detailed Description

template<typename Mesh_, int cell_dim_ = Mesh_::ShapeType::dimension>
struct FEAT::Geometry::ExportVTK< Mesh_, cell_dim_ >::UncompressedHeader

Header for uncompressed binary data.

Note that this header is mentioned nowhere in the VTK documentation, but is definitely required. For uncompressed data it contains the number of bytes of the following data (excluding the header itself) stored as a unsigned 32bit integer.
