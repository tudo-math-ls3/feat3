Detailed Description

template<typename DT_>
class FEAT::Analytic::Common::RigidBodyVortexVelo2D< DT_ >

Rigid-Body Vortex velocity field.

This class implements the Rigid-Body vortex velocity field, which is the velocity field of an analytical solution to the incompressible 2D Navier-Stokes equations on the unit-circle. The compatible pressure function is RigidBodyVortexPres2D.

\[v(x,y) := \begin{bmatrix}-y\\x\end{bmatrix}\]
