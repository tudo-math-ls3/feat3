Detailed Description

Helper-struct for storing relationships between meshpart entities.

The MeshPartOps class uses this to keep track of which indices refer to the same mesh entity in the left or right input meshparts of a meshpart op and the resulting output meshpart.

These relationships are determined for all entities of either of the input meshparts. Thus an entity might not have an index in the left or right meshparts. An entity is also not necessarily part of the output meshpart, and thus might not have an index in the output. Hence all members are optional.

The name JoinEntry is derived from join operations in databases.
