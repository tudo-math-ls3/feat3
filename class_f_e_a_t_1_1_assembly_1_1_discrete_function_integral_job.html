Detailed Description

template<typename Vector_, typename Space_, int max_der_>
class FEAT::Assembly::DiscreteFunctionIntegralJob< Vector_, Space_, max_der_ >

Assembly job for the integration of a discrete finite element function.

Template Parameters
Vector_The type of the coefficient vector of the discrete function. May be one of the following:
  • LAFEM::DenseVector<...>
  • LAFEM::DenseVectorBlocked<...>
Space_The finite element space used for the discretization.
max_der_The maximum derivative order of the integrals that are to be assembled.

Peter Zajac
