Detailed Description

template<typename DataType_, int k_ = 1>
class FEAT::Analytic::Common::SineTensorStatic< DataType_, k_ >

Sine-Tensor Static function.

This class implements the StaticFunction interface representing the function

  • 1D: u(x) = sin(k*pi*x)
  • 2D: u(x,y) = sin(k*pi*x) * sin(k*pi*y)
  • 3D: u(x,y,z) = sin(k*pi*x) * sin(k*pi*y) * sin(k*pi*z)

For any positive integral k, these functions are eigenfunctions of the Laplace operator. The corresponding eigenvalue is \( \lambda = -d (k\pi)^2 \), where d is the dimension of the domain.

Moreover, on any rectangular/rectoid domain [x0,x1]x[y0,y1]x[z0,z1] with integral domain boundaries xi,yi,zi, this function fulfills homogeneous Dirichlet boundary conditions.
