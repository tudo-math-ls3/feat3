Detailed Description

template<typename First_, typename... Rest_>
class FEAT::LAFEM::TupleMatrixRow< First_, Rest_ >

TupleMatrix row helper class template.

This class template is a helper class for the TupleMatrix class template, which is responsible for the management of a single "meta-row" of the outer tuple matrix.

Note
The implementation of this class is quite similar to that of TupleVector.
Template Parameters
First_,...Rest_A sequence of (meta) matrix classes which are to be composed.
Author
Peter Zajac
