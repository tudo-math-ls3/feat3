Detailed Description

template<typename DomainControl_>
class FEAT::Control::Meshopt::MeshoptControlBase< DomainControl_ >

Base class for Meshopt control objects.

Template Parameters
DomainControl_Type of the underlying domain control

For every mesh quality functional, there is a control class which inherits from this. This class only knows the bare minimum.
