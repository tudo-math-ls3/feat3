Detailed Description

template<typename SubType_, int count_>
class FEAT::LAFEM::PowerVector< SubType_, count_ >

Power-Vector meta class template.

This class template implements a composition of n sub-vectors of the same class.

Note
For a composed meta-vector of different sub-vector classes, see the TupleVector class template.
Template Parameters
SubType_The type of the sub-vector.
count_The number of sub-vector blocks.
Author
Peter Zajac
