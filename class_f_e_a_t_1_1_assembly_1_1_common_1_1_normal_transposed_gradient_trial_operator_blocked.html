Detailed Description

template<int dimension_>
class FEAT::Assembly::Common::NormalTransposedGradientTrialOperatorBlocked< dimension_ >

(gradu^T*n)*v operator implementation

Template Parameters
dimension_The dimension of the blocks of the FE functions

This class implements the weak formulation of the bilinear trace transposed gradient trial operator, i.e.

\[ \nabla \varphi^T \cdot n \psi \]

This class can be used with the BilinearOperator TraceAssembly class template to assemble a vector valued FE matrix.

Note
This operator expects the trafo eval object to have an assembled normal vector, which the wrapper traceassembly job needs to provide.
