Detailed Description

template<typename SubType_, int blocks_>
class FEAT::LAFEM::PowerDiagMatrix< SubType_, blocks_ >

Power-Diag-Matrix meta class template.

This class template implements a diagonal composition of n sub-matrices of the same class. This can be interpreted as a diagonal m-by-m matrix of other matrices.

Template Parameters
SubType_The type of the sub-matrix.
blocks_The number of sub-matrix blocks.
Author
Peter Zajac
