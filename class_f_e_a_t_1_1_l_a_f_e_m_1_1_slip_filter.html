Detailed Description

template<typename DT_, typename IT_, int BlockSize_>
class FEAT::LAFEM::SlipFilter< DT_, IT_, BlockSize_ >

Slip Filter class template.

Template Parameters
DT_Data type, i.e. double
IT_Index type, i.e. unsigned int
BlockSize_Size of the blocks, i.e. 2 for a filter acting on a velocity field in 2d

An application of this filter sets a given vector component, i.e. it sets the normal component of a given velocity vector to zero. This is for enforcing boundary conditions like

\[ u \cdot \nu = g ~\mathrm{on}~ \Gamma \]

in the strong sense. An application of the slip filter \( f_s \) is can be written as

\[ f_s(u,g) = u - (u \cdot \nu) \nu + g \nu. \]

In the Finite Element case, this boundary condition can only be enforced in the strong sense if the FE space \( V_h \) is at least \( \mathcal{C}^0 \). One possible approximation is the condition that

\[ \forall j \in \Gamma_h: \mathcal{N}_j(f_{s,h} \cdot \nu_h) = \mathcal{N}_j(g), \]

where \( \mathcal{N}_j \) is the node functional to the Dof j associated with \( \Gamma_h, f_{s,h} \) is a discrete slip filter mapping and \( \nu_h \) is some discrete approximation of \( \nu \). Assuming \( g = 0 \) for simplicity, if \(v_j\) is a basis function of \( V_h \), possible choice is

\[ f_{s,h}(v_j) = v_j - \frac{\mathcal{N}_j(v_j \cdot \nu_h)}{\mathcal{N}_j (\nu_h \cdot \nu_h)} \nu_h. \]

In the case of conforming Lagrange elements, this reduces to

\[ f_{s,h}(v_h)(x_j) = v_h(x_j) - \frac{v_h \cdot \nu_h}{\nu_h \cdot \nu_h}(x_j) \nu(x_j) \]

for all Lagrange nodes \( x_j \in \Gamma \).

At the moment, the SlipFilterAssembler is only implemented for Lagrange 1 functions and P1/Q1 transformations. For conforming Lagrange p spaces, interpolation of the outer unit normal field to the Lagrange p space needs to be implemented. For other function spaces (especially nonconforming ones) it not clear if the above approximation is sensible in any way.

Even for spaces with enough regularity (at least \( \mathcal{C}^0 \)), the application of Node functionals to FE functions of a different space is missing.

Also missing: Filtering of matrices.
