Detailed Description

template<typename LocalMatrix_, typename Mirror_, typename LocalFilter_, typename SolverBase_>
class FEAT::Solver::ADPSolverBase< Global::Matrix< LocalMatrix_, Mirror_, Mirror_ >, Global::Filter< LocalFilter_, Mirror_ >, SolverBase_ >

Implementation of ADPSolverBase interface for Global::Matrix instances.

This class implements the ADPSolverBase interface for Global::Matrix instances, which derive from standard finite element discretizations. Most of the dirty work is outsourced to the Global::AlgDofParti and Global::AlgDofPartiSystem classes.
