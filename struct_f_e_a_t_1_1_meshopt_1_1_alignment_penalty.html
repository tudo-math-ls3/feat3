Detailed Description

template<typename DT_, typename ShapeType_>
struct FEAT::Meshopt::AlignmentPenalty< DT_, ShapeType_ >

Wrapper class for functionality around aligning meshes to (implicit) surfaces.

Template Parameters
DT_The floating point type.
ShapeType_The mesh's shape type.

The alignment happens through using the distance function of the (implicit) surfaces and determining if it changes sign along an edge. If so, a penalty term is added to the functional value.
