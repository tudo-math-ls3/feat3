Detailed Description

template<typename DT_, int dim_>
class FEAT::Analytic::Common::YZPlaneParabolic< DT_, dim_ >

Velocity field for a parabolic profile in the y,z plane, constant in x.

Template Parameters
DT_The floating point type
dim_The space dimension

This is the mapping

\[ f: \mathbb{R}^d \to \mathbb{R}^d, f_0(x_0, ..., x_d)^T = \alpha \frac{4^{d-1}}{\prod_{i=1}^{d-1}(b_i - a_i)^2}\prod_{i=1}^{d-1}( (x_i - a_i)(b_i - x_i) ), f_i \equiv 0, i=1,\dots,d \]

for a given amplitude \( \alpha \). This means the function's \( x \)-component is \( \alpha \) at the midpoint of the rectangle \( \{x\}\times [a_1, b_1] \times [a_2, b_2] \) and zero on its boundary.
