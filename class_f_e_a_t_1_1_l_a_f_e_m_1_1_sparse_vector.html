Detailed Description

template<typename DT_, typename IT_ = Index>
class FEAT::LAFEM::SparseVector< DT_, IT_ >

Sparse vector class template.

Template Parameters
DT_The datatype to be used.
IT_The indexing type to be used.

This class represents a vector with non zero elements in a sparse layout.

Data survey:
_elements[0]: raw number values
_indices[0]: non zero indices
_scalar_index[0]: container size
_scalar_index[1]: non zero element count (used elements)
_scalar_index[2]: allocated elements
_scalar_index[3]: allocation size increment
_scalar_index[4]: boolean flag, if container is sorted
Refer to LAFEM design description for general usage informations.

Author
Dirk Ribbrock
