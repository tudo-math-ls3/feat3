Detailed Description

template<typename Mesh_, typename Lambda_>
class FEAT::Geometry::LambdaHitTestFactory< Mesh_, Lambda_ >

Lambda Hit-Test Factory class template.

This class template can be used to create a MeshPart for a particular mesh which consists of all entities that are inside the region characterized by a lambda expression hit-test function.

Template Parameters
Mesh_The type of the mesh for which the cell sub-set is to be computed.
Lambda_A lambda expression in the X/Y/Z coordinates returning a bool that specifies whether the tested point is inside the mesh-part region or not.

Peter Zajac
