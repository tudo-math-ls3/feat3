Detailed Description

template<typename Mesh_>
class FEAT::Geometry::ShapeConvertFactory< Mesh_ >

Shape-Conversion Mesh Factory class template.

This class template acts as a conversion factory for conformal Simplex and Hypercube meshes, i.e. this factory will convert a Simplex<n> mesh into a Hypercube<n> mesh or vice versa by conforming element subdivision.

Template Parameters
Mesh_The type that the input mesh is to be converted to.
