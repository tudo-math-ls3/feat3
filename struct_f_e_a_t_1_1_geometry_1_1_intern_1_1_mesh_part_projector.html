Detailed Description

template<typename AdaptiveMesh_, typename TargetMeshType_, typename Shape_>
struct FEAT::Geometry::Intern::MeshPartProjector< AdaptiveMesh_, TargetMeshType_, Shape_ >

Algorithm for projecting MeshParts of an underlying mesh onto its adaptive mesh.

This is the main declaration of the algorithm. The implementation is split across specializations for individual dimension to allow the algorithm to interact with meshes of any dimension.

The algorithm works in two phases. It first collects the references of all mesh entities that stem from one of the entites in the source meshpart. Then it writes a new meshpart using those references.
