Detailed Description

template<typename Coord_>
class FEAT::Geometry::MeshExtruderFactory< ConformalMesh< Shape::Hypercube< 2 >, 2, Coord_ > >

Mesh Extruder factory.

This class implements the Factory interface using a MeshExtruder object to extrude a given 2D quadrilateral mesh to a 3D hexahedral mesh.
