Detailed Description

template<typename SystemMatrix_, typename SystemFilter_, typename TransferOperator_>
class FEAT::Solver::MultiGridHierarchy< SystemMatrix_, SystemFilter_, TransferOperator_ >::LevelInfo

Multigrid level info class.

This helper class is used internally by the MultiGridHierarchy and MultiGrid classes to manage the various sub-solvers and temporary vectors.
