Detailed Description

template<typename DT_, typename IT_>
class FEAT::LAFEM::Container< DT_, IT_ >

Container base class.

All LAFEM-Container are derived from this base class.

See also
LAFEM design description
Template Parameters
DT_The datatype to be used.
IT_The indextype to be used.

This is the base class of all inheritated containers.

Data survey:
_scalar_index[0]: container size

Author
Dirk Ribbrock
