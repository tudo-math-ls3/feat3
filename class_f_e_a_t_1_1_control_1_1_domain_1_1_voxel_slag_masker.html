Computes the mask for an entire X-coordinate row.

Parameters
[in,out]maskThe mask vector for the current X-row. Is allocated to correct size, but its contents are undefined upon entry. Assuming that the length of this vector is n, then the i-th entry of the mask vector is to be set to 1 if the point with the X-coordinate equal to x_min + i/(n-1)*x_max is inside the mask, otherwise it is to be set to 0.
[in]x_min,x_maxThe minimal and maximal X-coordinate of the X-row for which the mask is to be computed.
[in]other_coordsThe point that contains the Y- and (in 3D) Z-coordinates of the X-row that for which the mask is to be computed. The first coordinate, i.e. point[0], is undefined, so only point[1] and (in 3D) point[2] are set to the Y-/Z-coords.
