Detailed Description

template<typename LambdaValue_, typename LambdaDx_ = void, typename LambdaDxx_ = void>
class FEAT::Analytic::LambdaScalarFunction1D< LambdaValue_, LambdaDx_, LambdaDxx_ >

Analytic 1D scalar lambda expression function implementation.

This class template acts as a wrapper class that implements the Analytic::Function interface for a 1D scalar function by using C++ lambda expressions for the actual formulae evaluation.

See also
Please refer to the Analytic Lambda Expression Functions for details about how to use this class.

Peter Zajac
