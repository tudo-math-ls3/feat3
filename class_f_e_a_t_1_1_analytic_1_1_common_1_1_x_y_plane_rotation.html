Detailed Description

template<typename DT_, int dim_>
class FEAT::Analytic::Common::XYPlaneRotation< DT_, dim_ >

Velocity field for a rigid body rotation in the x,y plane.

Template Parameters
DT_The floating point type
dim_The space dimension

This is the mapping

\[ f: \mathbb{R}^d \to \mathbb{R}^d, (x, y, z)^T \mapsto \omega (-(y - y_0), x - x_0, 0) \]

for a given angular velocity \( \omega \) and and origin \( (x_0, y_0, z_0)^T \).
