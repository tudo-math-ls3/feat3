Detailed Description

template<int dimension_>
class FEAT::Assembly::Common::LaplaceOperatorBlocked< dimension_ >

-Laplace operator implementation

This functor implements the weak formulation of the bilinear blocked Laplace operator, i.e.

\[ \nabla \varphi \cdot \nabla\psi \]

This functor can be used with the BilinearOperator assembly class template to assemble a blocked Laplace/Stiffness matrix.
