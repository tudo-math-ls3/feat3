Detailed Description

template<typename DT_>
class FEAT::Analytic::Common::Frankes3DVariantFunction< DT_ >

3D Version of Franke's Function

Template Parameters
DT_The floating point type
dim_The space dimension

For Franke's function f(x,y), this function represents \( g(x,y,z) = \exp(-(10*z-2)^2/25) \cdot f(x,y) \)
