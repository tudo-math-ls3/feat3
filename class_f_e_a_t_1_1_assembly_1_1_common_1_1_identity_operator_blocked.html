Detailed Description

template<int dimension_>
class FEAT::Assembly::Common::IdentityOperatorBlocked< dimension_ >

Vector-valued identity operator implementation.

This functor can be used with the BilinearOperator assembly class template to assemble one scalar matrix corresponding to one block of the block matrix, where is the number of space dimensions.
