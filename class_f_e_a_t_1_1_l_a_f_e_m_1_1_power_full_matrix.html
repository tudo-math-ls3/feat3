Detailed Description

template<typename SubType_, int width_, int height_>
class FEAT::LAFEM::PowerFullMatrix< SubType_, width_, height_ >

Power-Full-Matrix meta class template.

This class template implements a composition of m xn sub-matrices of the same class. This can be interpreted as an m-by-n matrix of other matrices.

Template Parameters
SubType_The type of the sub-matrix.
width_The number of sub-matrix blocks per row.
height_The number of sub-matrix blocks per column.
Author
Peter Zajac
