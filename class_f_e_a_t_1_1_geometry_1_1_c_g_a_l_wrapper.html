Detailed Description

template<typename DT_>
class FEAT::Geometry::CGALWrapper< DT_ >

Wrapper for the CGAL Library.

Template Parameters
DT_The dataype tp be used.

This class acts as a wrapper for a small portion of the CGAL Library. As of now, only the interaction with 3d tetrahedral OFF files is implemented.

Note
https://www.cgal.org/
