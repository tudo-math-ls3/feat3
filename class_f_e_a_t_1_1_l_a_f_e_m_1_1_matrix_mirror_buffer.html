Detailed Description

template<typename DT_, typename IT_ = Index>
class FEAT::LAFEM::MatrixMirrorBuffer< DT_, IT_ >

Matrix Mirror Buffer class template.

Template Parameters
DT_The datatype to be used.
IT_The indextype to be used.

This class holds all matrix mirror informations needed on the corresponding other rank.

The data layout resembles the common csr layout, with the exception of multiple data entries per non zero element.
Data survey:
_elements[0]: raw number values
_indices[0]: column index per non zero element
_indices[1]: row start index (including matrix end index)
_scalar_index[0]: container size _scalar_index[1]: row count
_scalar_index[2]: column count
_scalar_index[3]: non zero element count (used elements)
_scalar_index[4]: entries per nonzero count

Dirk Ribbrock
