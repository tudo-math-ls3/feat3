Detailed Description

template<typename Matrix_, typename Space_>
class FEAT::Assembly::TraceAssemblyJumpStabilizationMatrixJob< Matrix_, Space_ >

Matrix assembly job for the edge-oriented jump stabilization operator.

This class implements the TraceAssemblyJob interface to assemble the jump stabilization operator:

\[J(\varphi,\psi) = \gamma \sum_E (s\cdot J_E)^{p} \int_E [\nabla \varphi]\cdot[\nabla\psi]\]

Template Parameters
Matrix_The matrix that is to be assembled.
Space_The finite element space to be used as test- and trial-space.
