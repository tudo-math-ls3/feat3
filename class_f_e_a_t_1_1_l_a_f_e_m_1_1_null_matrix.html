Detailed Description

template<typename DT_, typename IT_, int BlockHeight_ = 1, int BlockWidth_ = 1>
class FEAT::LAFEM::NullMatrix< DT_, IT_, BlockHeight_, BlockWidth_ >

Null Matrix implementation.

This class implements a null matrix in the common LAFEM matrix interface. The main purpose of this class is to act as a "null block" inside a bigger meta-matrix, which is typically a LAFEM::TupleMatrix.

This matrix is a "hybrid-blocked-scalar" implementation, i.e. it can act as both a "blocked" null-matrix class as well as a "scalar" null-matrix class by setting BlockHeight_ = BlockWidth_ = 1. The compatible L/R vector types are chosen to be DenseVector or DenseVectorBlocked depending on whether the corresponding block dimension is 1 or greater than 1 (in analogy to the SparseMatrixBCSR class).

Also note that the DT_ and IT_ template arguments are only required for the conformance with the LAFEM matrix interface.
