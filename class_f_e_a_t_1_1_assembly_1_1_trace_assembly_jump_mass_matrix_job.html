Detailed Description

template<typename Matrix_, typename Space_>
class FEAT::Assembly::TraceAssemblyJumpMassMatrixJob< Matrix_, Space_ >

Matrix assembly job for the mass jump operator.

This class implements the TraceAssemblyJob interface to assemble the mass jump operator:

\[J(\varphi,\psi) = \alpha \sum_E \int_E [\varphi]\cdot[\psi]\]

Template Parameters
Matrix_The matrix that is to be assembled.
Space_The finite element space to be used as test- and trial-space.
