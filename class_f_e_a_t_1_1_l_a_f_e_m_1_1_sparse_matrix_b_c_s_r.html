Detailed Description

template<typename DT_, typename IT_, int BlockHeight_, int BlockWidth_>
class FEAT::LAFEM::SparseMatrixBCSR< DT_, IT_, BlockHeight_, BlockWidth_ >

CSR based blocked sparse matrix.

Template Parameters
DT_The datatype to be used.
IT_The indexing type to be used.

This class represents a sparse matrix, that stores its non zero elements in the compressed sparse row format. Every non zero element is a small dense matrix block on itself, represented as Tiny::Matrix objects. To be consistent with the external interface, the layout information are stored in block-scoped coordinates.

Data survey:
_elements[0]: raw non zero number values
_indices[0]: column index per non zero element
_indices[1]: row start index (including matrix end index)
_scalar_index[0]: container size
_scalar_index[1]: row count
_scalar_index[2]: column count
_scalar_index[3]: non zero element count (used ValueType elements)
Refer to LAFEM design description for general usage informations.

Adds the trace of a double matrix multiplikation to a vector.

This function performs the following computation:

\[ v \leftarrow v + \alpha \mathrm{tr}( D \textnormal{diag}(A) \cdot B\]

where \( B = \mathrm{this} \)

where

  • v denotes a m vector
  • D denotes a m-by-l matrix
  • A denotes a vector representing a l-by-l diagonal matrix
  • B denotes a l-by-n matrix

and the block sizes have to match accordingly:

  • v has a bm blocks
  • D has bm-by-bl blocks
  • A has bl blocks
  • B has bl-by-bn blocks
Note
This function currently only supports data in main memory.
Template Parameters
