Detailed Description

template<typename DataType_, int k_ = 1>
class FEAT::Analytic::Common::CosineTensorStatic< DataType_, k_ >

Cosine-Tensor Static function.

This class implements the StaticFunction interface representing the function

  • 1D: u(x) = cos(k*pi*x)
  • 2D: u(x,y) = cos(k*pi*x) * cos(k*pi*y)
  • 3D: u(x,y,z) = cos(k*pi*x) * cos(k*pi*y) * cos(k*pi*z)

For any positive integral k, these functions are eigenfunctions of the Laplace operator. The corresponding eigenvalue is , where d is the dimension of the domain.

Moreover, on any rectangular/rectoid domain [x0,x1]x[y0,y1]x[z0,z1] with integral domain boundaries xi,yi,zi, this function fulfills homogeneous Neumann boundary conditions including the integral-mean condition .
