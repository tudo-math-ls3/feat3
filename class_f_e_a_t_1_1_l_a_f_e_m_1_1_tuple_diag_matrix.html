Detailed Description

template<typename First_, typename... Rest_>
class FEAT::LAFEM::TupleDiagMatrix< First_, Rest_ >

Tuple-Diag-Matrix meta class template.

This class template implements a diagonal composition of n sub-matrices of arbitrary classes. This can be interpreted as a diagonal m-by-m matrix of other matrices.

Template Parameters
First_,...Rest_A sequence of (meta) matrix classes which are to be composed.
Author
Peter Zajac
