Detailed Description

template<typename T_>
class FEAT::Geometry::CGALValueIteratorWrapper< T_ >

Wrapper for arbitrary iterators that produce values.

Template Parameters
T_Value type

Works by accepting a std::function<std::optional<T_>()> as a generator function. All non-empty values produced by this function are output as iterator values. An empty value indicates no further values are to be produced. The function serves to hide the type of any underlying iterator.

Incrementing this iterator after the first empty value has been observed is forbiden.
