Detailed Description

template<typename DT_>
class FEAT::Analytic::Common::SineRingVortexPres2D< DT_ >

Sine-Vortex pressure function on the 2D ring domain with distance [1/2,1] around (0,0)

This function can be used as an analytic solution to the steady-state Stokes or Navier-Stokes equations on the 2D ring domain with distance [1/2,1] around the origin (0,0) when used in combination with the SineRingVortexVelo2D and SineRingVortexRHS2D functions for velocity and right-hand-side, respectively.
