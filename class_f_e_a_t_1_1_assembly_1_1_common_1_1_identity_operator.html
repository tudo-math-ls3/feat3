Detailed Description

Identity operator implementation.

This functor implements the weak formulation of the bilinear scalar Identity operator, i.e.

\[ \varphi \cdot \psi \]

This functor can be used with the BilinearOperator assembler class template to assemble a scalar mass matrix.
