Detailed Description

template<typename DataType_>
class FEAT::Analytic::Common::ParProfileScalar< DataType_ >

2D Scalar Parabolic Profile function

This class represents a scalar parabolic profile along a 2D line segment, which is given by the coordinates (x_0, y_0) and (x_1, y_1).

inlineinherited

Parses the profile parameters from a string.

This function can be used to parse the configuration for the parabolic profile from a string, which may e.g. be read from the command line or a text file.

The supported syntax for the string is

\[(x_0 y_0 , x_1 y_1)\]

or

\[(x_0 y_0 , x_1 y_1 , v_{max})\]

Parameters
[in]sbcThe string to be parsed.
Returns
true, if the string was parsed successfully, otherwise false.
