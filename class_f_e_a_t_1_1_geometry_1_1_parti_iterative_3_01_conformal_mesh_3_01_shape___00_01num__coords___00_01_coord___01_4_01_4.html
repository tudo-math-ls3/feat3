Detailed Description

template<typename Shape_, int num_coords_, typename Coord_>
class FEAT::Geometry::PartiIterative< ConformalMesh< Shape_, num_coords_, Coord_ > >

Iterative-Partitioner class template specialization for ConformalMesh.

The basic usage of this class is as follows:

  1. Refine the mesh until it contains at least num_patches cells.
  2. Create an object of this class and pass the to-be-partitioned mesh as well as the desired number of patches and the available number of ranks in a communicator to the constructor.
  3. Create the Elements-At-Rank graph using the build_elems_at_rank() function.
