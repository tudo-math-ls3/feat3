Detailed Description

template<typename Scalar_, typename DataType_>
class FEAT::Analytic::Common::TensorStatic< Scalar_, DataType_ >

Wrapper class for Tensor-Product scalar static functions.

This class can be used as a wrapper around another class implementing the StaticFunction interface for 1D. This class will then implement the 2D and 3D by using the tensor-product approach.

Template Parameters
Scalar_The name of the class implementing the StaticFunction interface for 1D.
DataType_The data-type used by Scalar_.
