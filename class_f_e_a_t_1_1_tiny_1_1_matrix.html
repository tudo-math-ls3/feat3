Detailed Description

template<typename T_, int m_, int n_, int sm_ = m_, int sn_ = n_>
class FEAT::Tiny::Matrix< T_, m_, n_, sm_, sn_ >

Tiny Matrix class template.

This class template implements a matrix whose value type and size is given at compile-time. The value type can be a primitive type, or some other object like a Vector.

Template Parameters
T_The datatype that the vector shall contain.
m_,n_The number rows and columns of the matrix. Must be > 0.
sm_The row stride of the matrix. Must be >= m_.
sn_The column stride of the matrix. Must be >= n_.
Author
Peter Zajac
