Detailed Description

template<typename TemplateSet_, typename Shape_, int dim_ = Shape_::dimension>
struct FEAT::Geometry::Intern::ElementChildren< TemplateSet_, Shape_, dim_ >

Children of a mesh element.

Template Parameters
TemplateSet_TemplateSet used for the mesh
Shape_The shape of the parent entity
dim_Current dimension of this data structure

Note that the size of this struct is determined at compile time and that this struct is stored inline with all other mesh element data. Using a template set with a large amount of children for any template will thus increase the memory usage of all mesh elements.
