Detailed Description

template<typename Mesh_>
class FEAT::Trafo::IsoSphere::Mapping< Mesh_ >

Iso-Sphere transformation mapping class template.

This transformation implements an iso-parametric transformation for the 2D unit-circle or the 3D unit-sphere domain.

Template Parameters
Mesh_The mesh class that this transformation is to be defined on. This must be a mesh that represents a simplex surface mesh.
Author
Peter Zajac
