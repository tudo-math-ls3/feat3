Detailed Description

template<typename DT_, typename IT_, int BlockSize_>
class FEAT::LAFEM::SparseVectorBlocked< DT_, IT_, BlockSize_ >

Sparse vector class template.

Template Parameters
DT_The datatype to be used.
IT_The indexing type to be used.
BlockSize_The size of the represented blocks

This class represents a vector with non zero element blocks in a sparse layout.
Logical, the data are organized in small blocks of BlockSize_ length.

Data survey:
_elements[0]: raw number values
_indices[0]: non zero indices
_scalar_index[0]: container size - aka block count
_scalar_index[1]: non zero element count (used elements)
_scalar_index[2]: allocated elements
_scalar_index[3]: allocation size increment
_scalar_index[4]: boolean flag, if container is sorted
Refer to LAFEM design description for general usage informations.

Author
Dirk Ribbrock
