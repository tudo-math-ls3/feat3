Detailed Description

template<typename TrafoType_, typename DataType_>
struct FEAT::Meshopt::RumpfTrafo< TrafoType_, DataType_ >

Computes quantities associated with the transformation to Rumpf reference cells.

Generic class template for RumpfTrafos.

Template Parameters
TrafoType_Transformation type, the ShapeType is essential

For details on the reference cells, refer to [Rum96].

static

Computes the optimal local mesh size.

The optimal local mesh size is calculated from the weights lambda_ such that the volumes of all cells add up to the domain's volume, if each local cell was transformed to a Rumpf reference element with the appropriate size.

Parameters
[in]h_Vector of local mesh sizes
[in]lambda_Vector of element weights
[in]sum_det_The sum of the volumes of all optimally scaled reference cells
