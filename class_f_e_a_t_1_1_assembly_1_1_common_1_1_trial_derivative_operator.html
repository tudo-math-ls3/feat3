Detailed Description

Trial-Derivative operator implementation.

This functor implements the weak formulation of the bilinear trial-function derivative operator, i.e.

\[ \partial_i \varphi \cdot \psi \]

This functor can be used with the BilinearOperator assembly class template to assemble a scalar matrix for the pressure-gradient operator of the Stokes equation.
