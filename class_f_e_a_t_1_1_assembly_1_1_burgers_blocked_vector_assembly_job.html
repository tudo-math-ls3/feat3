Detailed Description

template<typename Vector_, typename Space_, typename ConvVector_ = Vector_>
class FEAT::Assembly::BurgersBlockedVectorAssemblyJob< Vector_, Space_, ConvVector_ >

Burgers assembly job for block right-hand-side vector.

This assembly job implements the blocked (i.e. vector-valued) Burgers operator assembly, which is assembled into a right-hand-side vector.

See the documentation of the BurgersAssemblyJobBase class template (which is a base class of this class) for details about the terms and coefficients offered by this assembly job.

Template Parameters
Vector_The type of the blocked vector that is to be assembled.
Space_The finite element space to be used for assembly
ConvVector_The type of the convection (velocity) vector field; usually identical to Vector_
