Detailed Description

template<int dom_dim_, int img_dim_ = dom_dim_>
class FEAT::Analytic::ParsedVectorFunction< dom_dim_, img_dim_ >

Parsed vector function implementation.

This class provides an implementation of the Analytic::Function interface for vector functions which can parse and evaluate a formula tuple in the up to three variables 'x', 'y' and 'z' as well as user-defined extra variables given as a string at runtime.

Template Parameters
dom_dim_The domain dimension of the vector function. Must be 1 <= dom_dim_ <= 3.
img_dim_The image dimension of the vector function. Must be >= 1.

See the documentation of ParsedScalarFunction for more details.
