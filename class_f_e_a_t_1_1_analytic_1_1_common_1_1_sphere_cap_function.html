Detailed Description

template<int dim_, int a_ = 2>
class FEAT::Analytic::Common::SphereCapFunction< dim_, a_ >

Sphere-Cap Function on the 2D unit circle / 3D unit ball.

This function is given by the formula \( (\sqrt{a - \|x\|_2^2} - \sqrt{a-1}) \), which represents the upper half-sphere for a_ = 1. For a_ > 1, this function represents just a part of the upper half-sphere, which is regular even for \(\|x\|_2=1\).

It is recommended to use this function with a_ = 2, which yields a sufficiently regular function on the 2D/3D unit circle/ball.
