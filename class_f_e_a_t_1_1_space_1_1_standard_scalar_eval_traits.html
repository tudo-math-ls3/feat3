Detailed Description

template<typename EvalPolicy_, int max_local_dofs_, typename DataType_ = typename EvalPolicy_::DataType>
class FEAT::Space::StandardScalarEvalTraits< EvalPolicy_, max_local_dofs_, DataType_ >

Standard scalar evaluator traits class template.

This class implements a simple evaluator traits which should suffice for most scalar elements.

Template Parameters
EvalPolicy_The evaluation policy that is to be used. See Trafo::StandardEvalPolicy.
max_local_dofs_The maximum number of local degrees of freedom.
DataType_The data type that is to be used.
