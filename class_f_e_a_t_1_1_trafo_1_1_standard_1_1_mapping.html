Detailed Description

template<typename Mesh_>
class FEAT::Trafo::Standard::Mapping< Mesh_ >

Standard transformation mapping class template.

This class implements the standard first-order transformation mapping for any sort of mesh.

Template Parameters
Mesh_The mesh class that this transformation is to be defined on.
Author
Peter Zajac
