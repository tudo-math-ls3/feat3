Detailed Description

template<typename Space_, typename DT_, typename IT_>
struct FEAT::LAFEM::VectorGatherScatterHelper< Space_, DT_, IT_ >

Standalone Vector Gather and Scatter Axpy Interface.

This class provides standalone implementations for Gather- and Scatter-Axpy operations for different underlying vector containers. You have to extract the data arrays and choose the correct functions yourself (for now), so be sure these are correct. The functions are also provided as (cuda) device functions, if compiled by a nvcc compile unit. Furthermore, grouped operations, see cuda cooperative groups, are also available.

Template Parameters
Space_The underlying space.
DT_The datatype to be used.
IT_The indextype to be used.
Author
Maximilian Esser
