Detailed Description

template<typename DataType_, typename IndexType_>
class FEAT::LAFEM::PointstarFactoryFD2< DataType_, IndexType_ >

Finite-Differences pointstar matrix factory.

This class generates poinstar matrices in Finite-Differences style. Moreover, this class can compute the largest and smallest eigenvalue of the corresponding matrix as well as the eigenvector with respect to the smallest eigenvalue.

Computes the largest eigenvalue of the FD-style matrix.

The largest eigenvalue of the FD-style matrix is given as

\[ \lambda_{\max} = 2 \cdot \sum^d_{k=1} h_k^2 \big(1 + \cos(\pi h_k) \big) \]

Computes the smallest eigenvalue of the FD-style matrix.

The smallest eigenvalue of the FD-style matrix wrt the sine-bubble eigenvector is given as

\[ \lambda_{\min} = 2 \cdot \sum^d_{k=1} h_k^2 \big(1 - \cos(\pi h_k) \big) \]

Returns
The smallest eigenvalue of the matrix.
