Detailed Description

template<typename FineMesh_, typename CoarseMesh_ = FineMesh_, bool is_structured_ = FineMesh_::is_structured>
class FEAT::Geometry::Intern::CoarseFineCellMapping< FineMesh_, CoarseMesh_, is_structured_ >

A coarse cell to fine cell mapping class.

Template Parameters
FineMesh_The fine mesh type to be used.
CoarseMesh_The coarse mesh type to be used.
is_structured_Classification of the mesh types.

This class provides the mapping of coarse cells to fine cells.

Attention
This class silenty assumes that fine_mesh is the mesh that was obtained by applying the standard 2-level refinement algorithm on the coarse_mesh.
