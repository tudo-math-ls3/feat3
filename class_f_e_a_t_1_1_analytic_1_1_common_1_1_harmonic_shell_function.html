Detailed Description

template<int dim_, typename DataType_>
class FEAT::Analytic::Common::HarmonicShellFunction< dim_, DataType_ >

Harmonic shell function.

This class implements a harmonic function u, which fulfills u(x) = 1 for |x - x_0|_2 = 1 and u(x) = 2 for |x - x_0|_2 = 1/2.

This function is given by the following formulae:

  • 1D: u(x) := 3 - 2*|x-x_0|
  • 2D: u(x,y) := 1 - ln((x-x_0)^2 + (y-y_0)^2)/(2*ln(2))
  • 3D: u(x,y,z) := 1/sqrt((x-x_0)^2 + (y-y_0)^2 + (z-z0)^2)
Note
This function is harmonic and it is basically a scaled Green's function for the Laplace operator, therefore it can be used as an analytical solution for the Laplace equation on the unit ring/shell domain with Dirichlet boundary conditions u=1 for the outer boundary and u=2 for the inner boundary.
Warning
Because the function is differentiable everywhere except for x_0, Bad Things (TM) might happen if someone wants to compute the gradient or hessian there, as the functions return 0.
