Detailed Description

template<typename DataType_, typename IndexType_ = Index>
class FEAT::LAFEM::NoneFilter< DataType_, IndexType_ >

None Filter class template.

This class implements a filter that does nothing.

Note
Not to be confused with do-nothing boundary conditions.
