Detailed Description

template<typename Trafo_>
class FEAT::Space::CroRavRanTur::Element< Trafo_ >

Crouzeix-Raviart / Rannacher-Turek element class template.

This class template implemements the second-order H1-non-conforming finite element spaces known as the Crouzeix-Raviart (Simplex shapes) and Rannacher-Turek (Hypercube shapes).

Note
In the case of Hypercube meshes, this class implements the non-parameteric integral-mean based implementation, which is known to be the most stable of all variants.
See also
[CR73], [RT92]

template<typename Trafo_ >
constexpr int FEAT::Space::CroRavRanTur::Element< Trafo_ >::local_degree = Intern::LocalDegree<ShapeType>::value
staticconstexpr

Local Polynomial degree count.

This constant describes the maximum local polynomial degree of the element. This may be used, e.g. for automated determination of appropriate cubature rules.

Please note that for Simplex elements, 'degree' refers to the P_k spaces, whereas for Hypercube elements, 'degree' refers to the Q_k spaces.
