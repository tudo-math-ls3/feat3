Detailed Description

template<int dimension_>
class FEAT::Assembly::Common::GradientTrialOperatorBlocked< dimension_ >

Gradient operator applied onto the trial function.

This is the gradient bilinear form using blocked types without partial integration: \( a(\phi, \psi)_m = \int_\Omega(\nabla \phi, \psi e_m) dx, m=1,\dots,d \)
