Detailed Description

template<typename DT_>
class FEAT::Meshopt::ConcentrationFunctionPowOfDist< DT_ >

Default elemental distance concentration function.

Template Parameters
DT_Floating point precision

This maps a signed distance to a concentration according to

\[ c(d) = (\alpha + |d|)^\beta, \alpha \geq 0. \]
