Detailed Description

Standing-Vortex function in 2D.

This function represents the 2D standing vortex velocity field, which is a common benchmark velocity field for testing the stability of Navier-Stokes discretizations and stabilizations.

The field is given with \(r := \sqrt{(x-0.5)^2 \cdot(y-0.5)^2}\) as

\[u(x,y,z) = \begin{cases}\begin{bmatrix}5(y-0.5)\\ -5(x-0.5)\end{bmatrix},& r < 0.2\\ \begin{bmatrix}(2/r - 5)(y-0.5) \\ (-2/r + 5)(x-0.5)\end{bmatrix},& 0.2 \leq r < 0.4\\0, & \text{else}\end{cases}\]
