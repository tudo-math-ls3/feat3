Detailed Description

Transfer Assembler for VoxelDomainControl levels.

This class is used by applications to assemble transfer operators based on a mesh hierarchy that is created and organized by the VoxelDomainControl class.

Author
Peter Zajac

Assembles the blocked transfer operators for a system level.

Parameters
[in,out]sys_lvl_fA transient reference to the fine system level whose transfer operators are to be assembled.
[in]sys_lvl_cA transient reference to the coarse system level, if it exists as a physical level in the domain control of this process, otherwise a reference to sys_lvl_f, thus indicating that this process does not participate in the parent layer.
[in]virt_lvl_fA transient reference to the fine virtual domain level.
[in]virt_lvl_cA transient reference to the coarse virtual domain level.
[in]cubatureThe name of the cubature rule to be used for integration.
[in]truncSpecifies whether the truncation operator is also to be assembled along with the prolongation and restriction operators.
[in]shrinkSpecifies whether the transfer matrices are to be shrunk, i.e. whether all entries whose absolute value is smaller than 1e-3 times the largest absolute value are to be removed from the matrices.
[in]space_lambdaA lambda expression that takes a const reference to a DomainLevel_ and returns a const reference to the finite element space on that level.
[in]transfer_lambdaA lambda expression that takes a reference to a SystemLevel and returns a reference to the local LAFEM::Transfer object on that level.
[in]gate_lambdaA lambda expression that takes a const reference to a SystemLevel and returns a const reference to the blocked gate on that level.
[in]muxer_lambdaA lambda expression that takes a const reference to a SystemLevel and returns a const reference to the blocked coarse muxer on that level.
