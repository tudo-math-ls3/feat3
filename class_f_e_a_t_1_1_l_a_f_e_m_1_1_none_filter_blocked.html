Detailed Description

template<typename DataType_, typename IndexType_, int BlockSize_>
class FEAT::LAFEM::NoneFilterBlocked< DataType_, IndexType_, BlockSize_ >

Blocked None Filter class template.

This class implements a filter that does nothing.

Note
Not to be confused with do-nothing boundary conditions.
