Detailed Description

template<typename SubFilter_, int count_>
class FEAT::LAFEM::PowerFilter< SubFilter_, count_ >

PowerVector meta-filter class template.

This class template implements a composition of n sub-filters of the same type, which can be applied onto PowerVector meta-container objects.

Note
For a composed meta-filter of different sub-filters, see the TupleFilter class template.
Template Parameters
SubFilter_The type of the sub-filter.
count_The number of sub-filter blocks.

Peter Zajac
