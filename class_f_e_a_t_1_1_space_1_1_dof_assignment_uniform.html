Detailed Description

template<typename Space_, int shape_dim_, typename DataType_, template< typename Tag_, int dim_ > class DofTraits_, typename DofTag_>
class FEAT::Space::DofAssignmentUniform< Space_, shape_dim_, DataType_, DofTraits_, DofTag_ >

Uniform Dof-Assignment class template.

Template Parameters
Space_The finite element space this dof-assignment is used by.
shape_dim_The dimension of the shape that this dof-assignment refers to.
DataType_The data-type to be used by the dof-assignment.
DofTraits_A dof-traits class template that defines the number of dofs per dimension.
DofTag_A tag class that is passed as a first parameter to the DofTraits_ class template.
