Detailed Description

template<typename HitFunc_, typename Mesh_>
class FEAT::Geometry::HitTestFactory< HitFunc_, Mesh_ >

Hit-Test Factory class template.

This class template can be used to create a MeshPart for a particular mesh, which consists of all entities that are inside the region characterized by a hit-test function.

Template Parameters
HitFunc_A class implementing the HitTest-Function interface. See SphereHitTestFunction for an example.
Mesh_The type of the mesh for which the cell sub-set is to be computed.
Author
Peter Zajac, Stefan Wahlers
