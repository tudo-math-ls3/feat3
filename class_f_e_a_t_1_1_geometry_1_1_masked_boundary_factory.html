Detailed Description

template<typename ParentMesh_>
class FEAT::Geometry::MaskedBoundaryFactory< ParentMesh_ >

MaskedBoundaryFactory implementation.

This class is an extension of the BoundaryFactory class, which creates a mesh part containing the set of all unmasked boundary faces for a given mesh. Facets of the mesh can be masked by added them individually using the add_mask_facet() function or by adding an entire mesh part by using the #add_mask_mehspart() function. Once all masked facets/meshparts have been added, one has to call the compile() function before creating the mesh part using this factory.
