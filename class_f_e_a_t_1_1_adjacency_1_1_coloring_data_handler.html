Detailed Description

Datahandler for inverse coloring data.

This class is initialized by coloring data and also holds (if supported) coloring data on the gpu.

Warning
This class does not handle allocating and freeing device data by itself, i.e. you have to call the init_device and free_device function yourself.

Parameters
