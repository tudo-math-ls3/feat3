Detailed Description

Datahandler for inverse coloring data.

This class is initialized by coloring data and also holds (if supported) coloring data on the gpu.

Warning
This class does not handle allocating and freeing device data by itself, i.e. you have to call the init_device and free_device function yourself.

Template Parameters

Constructs the coloring data handle from a coloring data array, i.e. a vector of ints or a Coloring graph.

Parameters
[in]coloringSome array of integers, representing colors.
[in]hintOptionally, provide the numbers of colors. If negative, this is deduced automatically.
Warning
If hint is smaller then the maximum color number, this will lead to an allocation error.
