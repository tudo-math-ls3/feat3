Detailed Description

template<int dim_, typename DataType_>
class FEAT::Analytic::Distance::DistanceFunction< dim_, DataType_ >

Analytic distance function.

This class implements the AnalyticFunction interface representing the distance function \( f(x) =\| x - x_0 \|_2 \)

This class supports function values, gradient and hessians for all dimensions.

Warning
Because the function is differentiable everywhere except for x_0, Bad Things (TM) might happen if someone wants to compute the gradient or hessian there, as the functions return 0.
