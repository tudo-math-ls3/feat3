Detailed Description

template<typename BilinearOperator_, typename Matrix_, typename Space_>
class FEAT::Assembly::DomainAssemblyBilinearOperatorMatrixJob1< BilinearOperator_, Matrix_, Space_ >

Matrix assembly job for BilinearOperator and identical test-/trial-spaces.

This class implements the DomainAssemblyJob interface to assemble a bilinearform that is provided as a BilinearOperator implementation.

Template Parameters
BilinearOperator_The bilinear operator that is to be assembled
Matrix_The matrix that is to be assembled.
Space_The finite element space to be used as test- and trial-space.
