Detailed Description

template<typename Space_, template< typename Tag_, int dim_ > class DofTraits_, typename DofTag_>
class FEAT::Space::DofMappingUniform< Space_, DofTraits_, DofTag_ >

Uniform Dof-Mapping class template.

This class implements the Dof-Mapping interface for a uniform finite-element space, which has a fixed number of dofs for each entity dimension.

Template Parameters
Space_The finite-element space that this dof-mapping is used by.
Traits_A traits class template defining the number of dofs per entity dimension.
