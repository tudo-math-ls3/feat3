Detailed Description

template<typename DT_, typename IT_>
class FEAT::LAFEM::SparseMatrixFactory< DT_, IT_ >

Factory for SparseMatrix construction.

The purpose of this class is to make implementing CSR matrices very simple.
The data is stored in a map and can be transfered to a CSR matrix.

Template Parameters
DT_The data type to be used for the sparse matrix.
IT_The index type to be used for the sparse matrix.
