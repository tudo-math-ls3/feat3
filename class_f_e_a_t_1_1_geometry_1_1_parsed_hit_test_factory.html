Detailed Description

template<typename Mesh_>
class FEAT::Geometry::ParsedHitTestFactory< Mesh_ >

Parsed-Hit-Test Factory class template.

This class template can be used to create a MeshPart for a particular mesh, which consists of all entities that are inside the region characterized by a function formula string.

Template Parameters
Mesh_The type of the mesh for which the cell sub-set is to be computed.

Creates the ParsedHitTestFactory.

The formula dictates the Meshpart. Every point, for which the formula equals >= 0 is in a "inner" point weighted with 1. Otherwise the point will be weighted zero.
