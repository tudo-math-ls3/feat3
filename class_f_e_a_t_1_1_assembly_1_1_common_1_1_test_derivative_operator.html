Detailed Description

Test-Derivative operator implementation.

This functor implements the weak formulation of the bilinear test-function derivative operator, i.e.

This functor can be used with the BilinearOperator assembly class template to assemble a scalar matrix for the pressure-gradient operator of the Stokes equation.
